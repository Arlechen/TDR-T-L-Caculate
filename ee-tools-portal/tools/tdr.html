物理原理说明

TDR (时域反射计) 的原理类似于 PCB 上的"雷达"。它测量信号到达阻抗变化点并反射回来的往返时间。

L = (Vp × Δt) / 2

传播速度 (Vp) 由介质的等效相对介电常数 (εeff) 决定：

Vp = c / √εeff
